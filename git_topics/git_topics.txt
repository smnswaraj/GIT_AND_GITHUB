üåø Branches
Command	                  |     Description
--------------------------------------------------------------------------

üîÅ Syncing with Remote
Command	                                         |      Description
--------------------------------------------------------------------------

git push -u origin <branch>	                     |      Push branch to remote and set upstream

git fetch	                                     |      Fetch latest changes from remote (but not merge)
